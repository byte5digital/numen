## 6. FAQs for Positioning

**Q: How is this different from ChatGPT + a prompt?**  
A: ChatGPT uses generic training. Numen uses *your* persona + LLM config. That means every output is already tuned to your voice. No prompt engineering needed. No manual editing to fix tone.

**Q: Do I need to set up a persona first?**  
A: Yes. You configure your brand voice, tone, messaging pillars once (takes ~5 min). Then every repurposing operation uses that config. If you're already using Numen for other features, you're done—no extra setup.

**Q: What if I want to repurpose *differently* for different teams?**  
A: You can create multiple personas (e.g., "Technical writing voice" vs. "Marketing voice") and choose which to apply per repurposing job.

**Q: Is this social-only?**  
A: No. While other tools focus on Twitter/LinkedIn, Numen does 8 formats including technical docs (FAQ, Product page copy, Newsletter sections). True omnichannel.

**Q: Can I batch repurpose?**  
A: Yes. Process multiple pieces at scale and see cost estimates upfront.

**Q: What about content staleness?**  
A: Numen flags when repurposed content is getting old, so you know when to refresh or retire it.
